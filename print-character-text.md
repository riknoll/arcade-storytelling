# print character text

Prints some text relative to the camera at the bottom of the screen. Can also include a label for a character name.

```sig
story.printCharacterText("here's some text", "character name")
```

## Parameters

* **text**: a [string](/types/string) that contains the text to print to the screen
* **label**: a [string](/types/string) to display above the text as a label
